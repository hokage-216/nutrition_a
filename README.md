# nutrition_one

## Description

This project is a web application designed to help users track their health information and generate personalized meal plans based on their individual profiles. It utilizes technologies such as bcrypt, Express.js, Node.js, dotenv, Handlebars, MySQL, and Sequelize to securely store user information and generate dynamic content.

##  Features

* User Authentication: Secure user registration and login using bcrypt for password hashing.
* User Profile Management: Users can input and update their health information including email, password, first and last name, sex, height, weight, age, and activity level.
* Health Information Tracking: The application allows users to input and monitor their health metrics over time.
* Personalized Meal Plans: Based on the user's health information and goals, the system generates daily meal plans tailored to their specific needs and preferences.

##  Technologies Used
* Express.js: A web application framework for Node.js used for building the backend.
* Node.js: A JavaScript runtime environment used for server-side development.
* bcrypt: A library used for hashing passwords to enhance security.
* dotenv: A module used to load environment variables from a .env file into process.env.
* Handlebars: A templating engine for creating dynamic HTML content.
* MySQL: A relational database management system used for storing user data.
* Sequelize: An ORM (Object-Relational Mapping) for Node.js used for database management and interactions.

## Setup Instructions

1. Clone the repository to your local machine.
```bash
git clone https://github.com/hokage-216/nutrition_a
```
2. Navigate to the project directory.
```bash
cd nutrition_a
```
3. Install dependencies using npm or yarn.
```bash
npm install
```
4. Set up your MySQL database and configure the connection details in a new .env file.
